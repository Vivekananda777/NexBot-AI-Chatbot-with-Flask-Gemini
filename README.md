
# 🤖 NexBot – AI Chatbot with Flask & Gemini

NexBot is a real-time AI-powered chatbot web application built using Flask and Google Gemini API. It provides a smooth conversational interface with dynamic responses, markdown rendering, and chat history management.

---

## 🚀 Features

- 💬 Real-time chatbot interaction
- 🧠 Powered by Google Gemini Pro API
- 🎨 Modern dark UI with chat bubbles
- 📜 Markdown rendering for rich responses
- 🔄 Chat history tracking
- 🗑️ Clear chat functionality
- ⚡ Lightweight and fast Flask backend
